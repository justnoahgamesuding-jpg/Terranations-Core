# Terranations HUD Pack

This is a vanilla resource pack for the hotbar, XP bar, hearts, and hunger sprites.

Zip the contents of this folder so the zip root contains:

- `pack.mcmeta`
- `assets/`

Do not zip the parent `terranations_hud_pack` folder itself.

For Minecraft 1.21.1, `pack_format` is set to `34`.
