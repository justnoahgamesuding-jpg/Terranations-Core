# Terranations HUD Pack

This folder is the standalone vanilla resource pack example for Terra's HUD sprites.

Use it when you want a normal Minecraft resource pack zip instead of the ItemsAdder content flow.

## Build the zip

Zip the contents of this folder so the zip root contains:

- `pack.mcmeta`
- `assets/`

Do not zip the parent `terranations_hud_pack` directory itself.

## Scope

This pack contains the standalone HUD sprite set for:

- hotbar
- XP bar
- hearts
- hunger

## Version note

For Minecraft `1.21.1`, `pack_format` is `34`.

## Preferred path for live Terra servers

For the active in-game Terra HUD workflow, use the ItemsAdder content pack instead:

- `examples/itemsadder/contents/terra_quest_hud`
