# Terra Starter Gear Pack

This pack now contains two ItemsAdder items:
- `settlers_hatchet`
- `copper_pioneer_axe`

Both use custom handheld models so they render correctly in the player's hand.

Model files are stored in `contents/terra_starter_gear/models/item/` as required by ItemsAdder.

Install:
1. Copy this `terra_starter_gear` folder into `ItemsAdder/contents/`
2. Run `/iazip`
3. Reload ItemsAdder
