# Terranations HUD Pack

This is a standalone vanilla resource pack for the hotbar, XP bar, hearts, and hunger sprites.

It is separate from the ItemsAdder content pack in `examples/itemsadder/contents/terra_quest_hud`.
Use this folder when you want a normal Minecraft resource pack zip.
Use the ItemsAdder content folder when you want ItemsAdder to build and serve the in-game pack.

Zip the contents of this folder so the zip root contains:

- `pack.mcmeta`
- `assets/`

Do not zip the parent `terranations_hud_pack` folder itself.

For Minecraft 1.21.1, `pack_format` is set to `34`.
